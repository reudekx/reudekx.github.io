## 기본 문법

### 변수 선언
Python에서는 변수를 선언할 때 타입을 명시하지 않습니다:

```python
name = "철수"
age = 25
height = 175.5
is_student = True
```

### 코멘트 작성
코드에 대한 설명은 `#`를 사용합니다:

```python
# 이것은 주석입니다
name = "영희"  # 변수에 이름을 저장
```

## 자료형

### 1. 숫자형
* 정수 (`int`)
* 실수 (`float`)
* 복소수 (`complex`)

예시 코드:
```python
# 숫자형 예제
number = 42        # 정수
pi = 3.14159      # 실수
complex_num = 1+2j # 복소수

print(f"정수: {number}")
print(f"실수: {pi}")
```

### 2. 문자열 (str)
문자열 조작 예시:

| 메서드 | 설명 | 예시 |
|--------|------|------|
| `upper()` | 대문자로 변환 | `"hello".upper()` |
| `lower()` | 소문자로 변환 | `"WORLD".lower()` |
| `split()` | 문자열 분할 | `"a,b,c".split(',')` |

### 3. 리스트 (list)
```python
# 리스트 예제
fruits = ["사과", "바나나", "오렌지"]
numbers = [1, 2, 3, 4, 5]

# 리스트 조작
fruits.append("망고")    # 항목 추가
numbers.remove(3)       # 항목 제거
```

## 제어문

### 조건문
```python
def 성적_평가(점수):
    if 점수 >= 90:
        return "A"
    elif 점수 >= 80:
        return "B"
    elif 점수 >= 70:
        return "C"
    else:
        return "D"
```

### 반복문
1. for 루프:
```python
for i in range(5):
    print(f"{i}번째 반복")
```

2. while 루프:
```python
count = 0
while count < 3:
    print("파이썬 공부 중...")
    count += 1
```

## 함수

함수는 다음과 같이 정의합니다:

```python
def 인사말(이름, 나이=20):
    """
    인사말을 출력하는 함수
    
    매개변수:
        이름 (str): 사용자 이름
        나이 (int): 사용자 나이 (기본값: 20)
    """
    message = f"안녕하세요! 저는 {이름}이고, {나이}살입니다."
    return message

# 함수 호출
print(인사말("민수", 25))
```

### 예외 처리
```python
try:
    number = int(input("숫자를 입력하세요: "))
    result = 100 / number
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다!")
except ValueError:
    print("올바른 숫자를 입력하세요!")
else:
    print(f"결과: {result}")
```

### 문제 1 답안
```python
def sum_list(numbers):
    return sum(numbers)  # 또는 직접 구현: return reduce(lambda x,y: x+y, numbers)
```

### 문제 2 답안
```python
def is_palindrome(text):
    clean_text = ''.join(char.lower() for char in text if char.isalnum())
    return clean_text == clean_text[::-1]
```
