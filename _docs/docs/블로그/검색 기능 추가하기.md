# 검색 기능 추가하기

## 개요

블로그에 검색 기능을 추가해 보자.

## 방법

어떻게 추가할 것인지 생각해보자.

* 구글의 [Programmable Search Engine](https://programmablesearchengine.google.com)를 이용한다.
* 문서들의 정보를 담은 json 파일 등을 생성하여 클라이언트 쪽에서 검색한다.

전자의 경우 구글의 검색 엔진에 의존해야 하고, 
특히 UI를 원하는 대로 커스터마이징 하기 어렵다는 제약이 있다고 한다.

또 생각해 보면 어차피 혼자 사용하는 블로그인데.. 제목이나 카테고리 정도로만 검색이 가능하면 되지 않을까 싶다.
따라서 일단은 두 번째 방법으로 구현하는 것으로 하고,
json 형식으로 정보를 관리할지.. 등 어떻게 구현할 것인지는 차차 생각해 보도록 하자.
