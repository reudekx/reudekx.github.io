# 블로그

## 개요

블로그 개발과 관련된 문서들입니다.

## 개발 예정 사항

* a 태그 관련
    * 타이틀 속성 추가하기 (JS 파싱하여 빌드 시 삽입)
    * 외부 링크는 새 창에서 열리도록 하기 (JS 대신 파싱하여 빌드 시 삽입)  
    `<a href="https://example.com" target="_blank" rel="noopener noreferrer">`
* 문서 카테고리 추가하기
* [검색 기능 추가하기](/docs/블로그/검색 기능 추가하기)
* [메인](/) 페이지에 최근 수정된 문서 목록 표시하기
* [루트](/docs) 문서에 어떤 정보 나타낼지 기획하기
* 디렉터리 역할만 수행하는 문서들이 있는데, 해당 페이지들을 위한 레이아웃을 따로 만들어야 할지 고민 중
    * 가령 하위 문서 목록을 나타낼 수 있는 기능을 추가
        * 기존 마크다운 문법과 충돌하지 않도록 기호를 몇개 골라 커스텀 문법을 만들어서 넣어도 될 듯
