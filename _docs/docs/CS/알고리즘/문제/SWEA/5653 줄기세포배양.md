# SWEA 5653. 줄기세포배양

## 문제

* [https://swexpertacademy.com/main/code/problem/problemDetail.do?contestProbId=AWXRJ8EKe48DFAUo](https://swexpertacademy.com/main/code/problem/problemDetail.do?contestProbId=AWXRJ8EKe48DFAUo)

## 풀이

```python
# 상하좌우
DIRS = (
    (0, -1), (0, 1), (-1, 0), (1, 0)
)

# 세포
class Cell:
    # 생성자
    def __init__(self, life, age, was_here):
        self.life = life
        self.age = age
        self.was_here = was_here

    # 디버깅 시 출력 위해 정의
    def __repr__(self):
        return f"Cell({self.life}, {self.age}, {self.was_here})"

    # 활성화 여부 반환
    def is_active(self):
        return self.age >= self.life and self.age < self.life * 2
    
    # 생존 여부 반환
    def is_alive(self):
        return self.age < self.life * 2

# 세포 하나가 4방향으로 번식
def reproduce(pos, cell):
    for dy, dx in DIRS:
        ny, nx = pos[0] + dy, pos[1] + dx
        if (ny, nx) in cells:
            # 해당 위치에 있는 세포가 원래 있던 세포면 skip
            if cells[(ny, nx)].was_here:
                continue
            # 현재 세포의 생명력 수치가 더 높으면 덮어씌운다.
            if cells[(ny, nx)].life < cell.life:
                cells[(ny, nx)] = Cell(cell.life, 0, False)
        else:
            # 빈 셀이면 점유한다.
            cells[(ny, nx)] = Cell(cell.life, 0, False)

# 한 턴을 진행시킨다.
def process_turn():
    # 번식 시 dictionary가 수정되므로 copy하여 이용
    items = cells.copy().items()
    for pos, cell in items:
        if cell.is_active():
            reproduce(pos, cell)
        cell.age += 1

    for cell in cells.values():
        cell.was_here = True

# time 만큼 진행
def process(time):
    for _ in range(time):
        process_turn()

# 살아있는 세포의 수를 센다.
def count():
    total = 0
    for cell in cells.values():
        if cell.is_alive():
            total += 1
    return total

T = int(input())

for i in range(T):
    N, M, K = map(int, input().split())
    init = [list(map(int, input().split())) for _ in range(N)]
    cells = {}
    for y in range(N):
        for x in range(M):
            if not init[y][x]:
                continue
            cells[(y, x)] = Cell(init[y][x], 0, True)
    process(K)
    print(f"#{i + 1} {count()}")
```

문제를 풀며 혹은 푼 이후 생각했던 점은 다음과 같다.

* 턴 시작 시 새로운 dictionary를 생성한 뒤 턴 진행 후 기존 dictionary를 덮어 씌울지 고민하였음.
    * 문제 요구사항이 그리 복잡하지 않아 Cell 클래스에 bool 변수 하나를 추가하는 방법을 선택하여 번식을 처리할 수 있었음.
^
* Cell의 age를 언제 증가시키고, 활성화 및 생존은 언제부터 언제까지인지 설정하는 것이 약간 까다로웠음.
    * 가령 턴마다 새로 생성된 Cell은 해당 턴 종료 후 age를 증가시키면 안 되었다.
        * 따라서 기존 Cell들을 순회하며 번식과 동시에 해당 Cell의 age를 1 증가시켰다.
        * 다만 효율은 떨어질 수도 있겠지만, 번식 이후 마지막에 모든 Cell을 순회하며 was_here을 True로 업데이트할 때
            was_here의 기존 값이 True인 Cell이 곧 지난 턴에 생성되었던 Cell이므로, 이때 해당 셀의 age를 증가시키는 것이
            논리상 더 자연스러웠을 것 같다.
