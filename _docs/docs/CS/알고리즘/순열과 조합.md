# 순열과 조합

## 개요

**순열**과 **조합**을 구현하는 법을 알아보자.

파이썬의 경우 `itertools` 모듈을 이용할 수도 있지만,
다른 언어로 코딩 테스트를 보거나
해당 모듈을 사용할 수 없는 경우를 대비하여
직접 구현할 줄 알아야 한다.

논리 자체는 간단하기 때문에 한번 숙지하고 나면
모듈의 사용법을 외우는 것보다 기억하기 더 쉬울 수도 있다.

## 코드

함수를 리스트에 적용할 때
리스트에 중복된 값을 가진 원소가 있어도 index가 다르므로 구분된 원소로 취급함에 유의하자.

즉 **선택의 대상이 index**임을 인지하며 코드를 읽으면
이해가 쉬울 것이다.

### 순열

* **k개의 원소를 중복 없이, 순서를 고려하여 선택한다.**
* `used` 배열을 사용하여 중복 선택을 방지한다.

```python
def permutations(arr, k):
    used = [False for _ in arr]
    chosen = []

    def generate():
        if len(chosen) == k:
            yield chosen[:]
            return

        for i in range(len(arr)):
            if used[i]:
                continue
            chosen.append(arr[i])
            used[i] = True
            yield from generate()
            used[i] = False
            chosen.pop()

    return generate()
```

### 조합

* **k개의 원소를 중복 없이, 순서를 고려하지 않고 선택한다.**
* 한 원소를 선택한 후 그 다음 인덱스 **초과**의 범위 내에서 다음 원소를 선택한다.
* 모순적이게도 **순서를 고려하지 않는다는 점을 구현**하기 위해
    **오름차순 순서를 고려하여 선택**하도록 코드를 작성해야 한다.

```python
def combinations(arr, k):
    chosen = []

    def generate(start = 0):
        if len(chosen) == k:
            yield chosen[:]
            return
        
        for i in range(start, len(arr)):
            chosen.append(arr[i])
            yield from generate(i + 1)
            chosen.pop()
        
    return generate()
```

### 중복 순열

* **k개의 원소를 중복을 허락하여, 순서를 고려하며 선택한다.**
* 조건 없이 `chosen`을 채워 완성한다.

```python
def repeated_permutations(arr, k):
    chosen = []

    def generate():
        if len(chosen) == k:
            yield chosen[:]
            return
        
        for i in range(len(arr)):
            chosen.append(arr[i])
            yield from generate()
            chosen.pop()

    return generate()
```

### 중복 조합

* **k개의 원소를 중복을 허락하여, 순서를 고려하지 않고 선택한다.**
* 한 원소를 선택한 후 그 다음 인덱스 **이상**의 범위 내에서 다음 원소를 선택한다.
* 조합과 마찬가지로 **오름차순 순서를 고려**하며, 중복 선택이 가능하도록 코드를 작성해야 한다.

```python
def repeated_permutations(arr, k):
    chosen = []

    def generate(start = 0):
        if len(chosen) == k:
            yield chosen[:]
            return
        
        for i in range(start, len(arr)):
            chosen.append(arr[i])
            yield from generate(i)
            chosen.pop()
        
    return generate()
```
