# 외워서 끝내는 네트워크 핵심이론

## 개요

인프런의 **널널한 개발자** 강사님께서 진행하는
외워서 끝내는 네트워크 핵심이론
강의를 들으며 학습한 내용을 정리하였다.

강의 링크는 다음과 같다.

* [외워서 끝내는 네트워크 핵심이론 - 기초](https://www.inflearn.com/course/%EB%84%A4%ED%8A%B8%EC%9B%8C%ED%81%AC-%ED%95%B5%EC%8B%AC%EC%9D%B4%EB%A1%A0-%EA%B8%B0%EC%B4%88)
* [외워서 끝내는 네트워크 핵심이론 - 응용](https://www.inflearn.com/course/%EB%84%A4%ED%8A%B8%EC%9B%8C%ED%81%AC-%ED%95%B5%EC%8B%AC%EC%9D%B4%EB%A1%A0-%EC%9D%91%EC%9A%A9)

학습 내용을 블로그에 업로드하는 것에 대해서는
강사님께 허락을 받았고,
저작권 관련 문제가 발생하지 않도록
최대한 주의하고 있다.

또한 강의에 등장한 내용 뿐 아니라 내가 **임의로 추가한 내용**도
존재하므로 유의하길 바란다. (즉 **잘못된 정보**가 서술되어 있을 수 있다.)

누군가 내가 정리한 내용을 보고 강의에 흥미가 생긴다면
직접 수강을 하며 자신만의 공부를 하는 것을 적극 권장한다.
